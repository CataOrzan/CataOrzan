5DOF Robotic Arm

  • Category: Programming
  • Type: University project
  • Project date: 8 June 2023
  • Project Files: Project report | CAD Model | Matlab - Simulink model |

 

The project consists of building a robot with 5 degrees of freedom, which also involves implementing direct and inverse kinematics in MATLAB. A model was built and sized 3D model of the robot using CATIA V5R19. Two motors for the Y-axis rotational coupling. The distance between the robot couplings is 200mm. The robot is driven by motors with a snail wheel system, already existing on the market, powered by 12V.
